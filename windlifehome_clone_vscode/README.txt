Wind Life Home - clone local para VS Code

Como abrir:
1. Extraia o ZIP.
2. Abra a pasta no VS Code.
3. Abra o arquivo index.html com a extensão Live Server / Go Live.

Observações:
- As imagens que estavam embutidas no HTML foram extraídas para assets/img.
- A fonte Montserrat ficou via Google Fonts por link CDN. Não incluí arquivos de fonte no ZIP.
- Se alguma foto do site original não aparecer, é porque ela não veio dentro do HTML enviado.
- O WhatsApp está com o número placeholder que estava no HTML original: 5585999999999.
